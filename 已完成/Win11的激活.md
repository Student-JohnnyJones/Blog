# Win11的激活

1. Win + R，输入powershell，打开powershell
2. 运行以下命令：

```powershell
irm https://massgrave.dev/get | iex

# ps会弹出警告，说这条命令将被弃用，如果以后真的被弃用了，换成以下命令
irm https://get.activated.win | iex
```

3. 在弹出的窗口输入1，等待程序执行完成
4. 按任意键退出，按0退回powershell
5. 进入设置 -> 系统 -> 激活，检查激活状态



**Ps:** 这里我们还贴上一个网站：https://massgrave.dev。它里面有关于激活windows系统的更详细的描述，出现问题了就往这边找
