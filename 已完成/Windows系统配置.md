# Windows配置

## 前置操作

完成系统的安装和**激活**，如果没有激活，某些配置无法执行

## 系统配置

1. **熄屏配置**：系统 -> 电源 -> 屏幕、睡眠和休眠超时 -> 在此时间后关闭我的屏幕，选择“从不”
2. **设置深色模式**：个性化 -> 颜色 -> 选择模式：深色
3. **关闭PIN**：账户 -> 登录选项 -> 为了提高安全性……使用 Windows Hello 登录(推荐)，关闭。Win R运行：netplwiz。取消勾选“要运行本计算机，用户必须输入用户名和密码”，输入微软密码（注意是微软密码，不是pin），重启即可。
4. **开启sudo**：系统 -> 开发者选项 -> 启用sudo
5. **默认打开此电脑**：打开任意文件夹 -> 点击上方“**···**” -> 选项 -> 更改“打开文件资源管理器时打开”为“此电脑”
6. **显示后缀名**：打开任意文件夹 -> 点击上方“**···**” -> 选项 -> 查看 -> 在很下面找到“隐藏已知文件类型的后缀名” -> 取消勾选 -> 应用 -> 应用到文件夹

## 桌面配置

1. **显示此电脑**：系统 -> 个性化 -> 主题 -> 桌面图标设置，勾选“计算机”

2. **更改桌面路径**：进入此电脑，在左边框内找到“桌面”，右键选择“属性” -> 位置 -> 移动，在D盘中创建Desktop文件夹，选择Desktop，确定即可（注意千万不要直接将D盘作为桌面，否便样衰了，因为似乎会出bug）

3. **设置任务栏**：在任务栏右键，选择任务栏设置，关闭“任务视图”，“小组件”，“搜索”改成“隐藏”，任务栏行为改成靠左

4. **关闭桌面“了解此图片”**：个性化 -> 背景 -> 个性化设置背景：将windows聚焦改成随便别的什么选项即可

5. **更改存储路径**：系统 -> 存储 -> 高级存储设置 -> 保存新内容的地方：全部改成D: 

6. **配置开始菜单**：按win键打开开始菜单，右键底部空白处，进入设置

   - 选择“更多固定项”
   - 所有选项全关
   - “文件夹”：开启“设置”和“文件资源管理器”

7. **配置桌面**：

   - 右键，查看，勾选“自动排列图标” “将图标与网格对齐”
   - 创建“Downloads” “Temp” “Const”
   - 之后下载路径都更改到Downloads文件夹
